# Meme Maker

A light, fun, and responsive **Meme Generator** built with **React**.  
Create memes instantly using popular templates, add custom text, and download your meme as an image — all directly in the browser.


## Live Demo
https://mememakerpro.netlify.app


## Features

-  Light and playful UI designed for meme creation
-  Fully responsive and mobile-first layout
-  Random meme template generation using the Imgflip API
-  Custom top and bottom text input
-  Classic Impact-style meme font with bold black outline
-  Text automatically aligned at top and bottom of the image
-  Download generated memes as PNG images
-  No backend required — runs fully on the client

## Built With

- React
- CSS (Flexbox & Responsive Design)
- Imgflip API (for meme templates)
- html2canvas (for image download)
